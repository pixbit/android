Application Tutorial

Bookmarks–

Setting a bookmark - you can set bookmarks in the Regulations area only while you are in portrait view.

Navigate to where you would like to set the bookmark.

Tap once on the book icon in the upper right of the search bar.

Enter in the bookmark title in the dialog box and then save.

To navigate back to an area use the Bookmarks button and tap on the bookmark you want to navigate to.

To delete a bookmark use the Bookmark button, tap edit and then delete the bookmark.
